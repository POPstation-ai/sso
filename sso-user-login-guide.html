SSOログイン手順（ユーザー向け）

対象: POPstation を利用する一般ユーザー

目的: 自社の Microsoft アカウントで POPstation にログインする

目次

  1. SSOログインとは
  2. ログイン手順
  3. ログアウトと再ログイン

SSOログインとは

普段業務で使っている Microsoft アカウント（Microsoft 365 のアカウント等）を使って POPstation にログインできる仕組みです。

  • POPstation 専用のパスワードを覚える必要がありません
  • 既に Microsoft アカウントにログインしていれば、ボタン 1 つで POPstation にもログインできます
  • 多要素認証（MFA）が設定されている場合は、Microsoft アカウントの認証画面でそのまま実施します

ログイン手順

1. SSOログインURLを開く

会社（テナント管理者）から配布された SSO ログイン URL をブラウザで開いてください。URL の形式は以下のようになります:

https://popstation.ai/signin?sso=<英数字の文字列>

Tips: SSO ログイン URL をブラウザのブックマークに登録しておくと毎回のログインが楽になります。

2. Microsoftのサインイン画面でログイン

ログイン URL を開くと、自動的に Microsoft のサインイン画面に遷移します。

  1. 会社の Microsoft アカウント（メールアドレス）を入力 → 「次へ」
  2. パスワードを入力 → 「サインイン」
  3. 多要素認証が設定されている場合は、画面の指示に従って認証（スマートフォンアプリ承認・SMS コード入力など）
  4. 「サインインの状態を維持しますか？」と表示された場合はご自身の判断で選択

3. POPstation が表示される

サインインに成功すると、自動的に POPstation のチャット画面に遷移します。これでログイン完了です。

事前準備: SSO ログインを利用するには Microsoft アカウントとは別に POPstation 側にもユーザー登録が必要 です（管理者が事前に実施）。「Access Denied」と表示される場合は管理者に POPstation 側のユーザー登録を依頼してください。

ログアウトと再ログイン

ログアウト

POPstation の画面右上のメニューから「ログアウト」を選択してください。

再ログイン

再度ログインする際も、最初に開いた SSO ログイン URL を使ってください。ブックマークしておくと便利です。

すでに同じブラウザで Microsoft アカウントにサインイン中の場合は、サインイン画面が表示されずそのまま POPstation が開く場合があります。

困ったときは

Microsoft のサインイン画面で「アカウントを追加できませんでした」と表示される

SSO 連携対象外の Microsoft アカウント（個人用アカウント等）でサインインしようとしている可能性があります。会社の Microsoft アカウントでサインインしてください。

サインインしても POPstation に戻ってこない / 「Access Denied」エラーが出る

最も多い原因は POPstation 側のユーザー登録漏れ です。

  1. Microsoft 認証は成功しているが POPstation 側に当該メールアドレスのユーザーが未登録 → 管理者に POPstation 側のユーザー登録 を依頼
  2. ブラウザを完全に閉じて、再度 SSO ログイン URL を開く
  3. シークレットウィンドウ（プライベートブラウジング）で SSO ログイン URL を開く
  4. それでも解決しない場合は、自社のテナント管理者またはサポート窓口へお問い合わせください

問い合わせの際は、可能であれば以下の情報を添えてください:

  • エラー画面のスクリーンショット
  • 表示されたエラーメッセージ（特に AADSTS で始まる番号があれば原因特定が早まります）
  • 利用しているブラウザの種類とバージョン

通常のログイン画面でログインしたい

SSO ログイン URL ではなく、POPstation のトップ URL を開くと通常のログイン画面が表示されます。SSO 利用が必須かどうかは社内ポリシーに従ってください。

パスワードを忘れた

SSO ログインでは POPstation 用のパスワードは使用しません。Microsoft アカウントのパスワードを忘れた場合は、社内の IT 管理者にパスワードリセットを依頼してください。
